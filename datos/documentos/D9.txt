Las redes de computadores conectan dispositivos para compartir recursos e informacion. Los protocolos definen reglas para la comunicacion entre sistemas. El modelo TCP/IP estructura la comunicacion en capas funcionales. Las direcciones IP identifican unicamente cada dispositivo en la red. El enrutamiento determina el camino optimo para enviar paquetes de datos. Los switches y routers forwarding trafico segun las direcciones destino. El protocolo HTTP facilita la transferencia de documentos web. Los protocolos de seguridad como HTTPS cifran las comunicaciones. Las VPN crean tuneles seguros sobre redes publicas. El DNS traduce nombres de dominio a direcciones IP numericas. Los firewalls filtran trafico malicioso y no autorizado. La calidad de servicio prioriza trafico critico sobre el menos importante. Las redes inalambricas utilizan ondas radioelectricas para comunicarse. Los protocolos de routing dinamico adaptan rutas a cambios topologicos. La administracion de red monitorea y mantiene el funcionamiento optimo.
